LFSR_Signin

题目

思路&exp

非常简单的LFSR，给出flag长度为255，所以在模2下有以下等式：

x[i+255]=x[i]+x[i+20]+x[i+25]+x[i+250]+x[i+254] x[i+255] = x[i] + x[i+20] + x[i+25] + x[i+250] + x[i+254] x[i+255]=x[i]+x[i+20]+x[i+25]+x[i+250]+x[i+254]

变换得：

(x[i]+x[i+255])+x[i+255]=x[i]+x[i+20]+x[i+25]+x[i+250]+x[i+254]+(x[i]+x[i+255])即x[i]=x[i+20]+x[i+25]+x[i+250]+x[i+254]+x[i+255] (x[i] + x[i+255]) + x[i+255] = x[i] + x[i+20] + x[i+25] + x[i+250] + x[i+254] + (x[i] + x[i+255])\\ 即 x[i] = x[i+20] + x[i+25] + x[i+250] + x[i+254] + x[i+255] (x[i]+x[i+255])+x[i+255]=x[i]+x[i+20]+x[i+25]+x[i+250]+x[i+254]+(x[i]+x[i+255])即x[i]=x[i+20]+x[i+25]+x[i+250]+x[i+254]+x[i+255]

根据生成的值即可还原flag。

part2 & Siesta's revenge

这部分我本来是想考类似于extend wiener attack的造格方法的，但是因为flag取的太短导致了第一次的非预期，然后第二次s取的有点问题导致了第二次的非预期（烂完了）但是还是讲一下预期解的思路 我们有：

a1X1≡b1  (mod  P+s)a2X2≡b2  (mod  P+s) a_1X_1\equiv b_1 \;(mod\;P+s)\\ a_2X_2\equiv b_2 \;(mod\;P+s) a1​X1​≡b1​(modP+s)a2​X2​≡b2​(modP+s)

拆开得到：

a1X1+k1P=b1−k1s  ①a2X2+k2P=b2−k2s  ② a_1X_1 + k_1P = b_1 - k_1s\;①\\ a_2X_2 + k_2P = b_2 - k_2s\;② a1​X1​+k1​P=b1​−k1​s①a2​X2​+k2​P=b2​−k2​s②

然而由于未知数的个数过多，难以像part1那样构造线性的方程，因此需要想方法构造更多等式。我们可以先看一下①*②的式子如何：

①∗②=X1X2a1a2+X1Pa1k2+X2Pa2k1+P2k1k2注意到未知数仅有a1a2,a1k2,a2k1,k1k2四个，且对于①∗k2和②∗k1，构造的未知数不会比上式多。 ①*② = X_1X_2a_1a_2 + X_1Pa_1k_2 + X_2Pa_2k_1 + P^2k_1k_2\\ 注意到未知数仅有a_1a_2,a_1k_2,a_2k_1,k_1k_2四个，且对于①*k_2和②*k_1，构造的未知数不会比上式多。\\ ①∗②=X1​X2​a1​a2​+X1​Pa1​k2​+X2​Pa2​k1​+P2k1​k2​注意到未知数仅有a1​a2​,a1​k2​,a2​k1​,k1​k2​四个，且对于①∗k2​和②∗k1​，构造的未知数不会比上式多。
